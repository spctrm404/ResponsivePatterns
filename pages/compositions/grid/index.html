Composition: Grid

Grid는 클래스가 적용된 요소의 자녀요소들을 가로로 배열하되, 넘치면 다음 줄에 배열한다.

한 줄에 몇 개의 자녀요소가 배치될지는 부모요소의 컨텐츠 너비(W)와 자녀요소의 최소너비(m)에 따라 결정된다. 예를 들어 W=1000px, m=300px라면 한 줄에 3개의 자녀요소가 배치된다(1000 / 300 = 3.33). 이 때 1000을 300으로 나눈 나머지 100은 각 자녀요소에 균등하게 분배되어 각 자녀요소의 실제 너비는 300 + (100 / 3) = 333.33px이 된다.

만약 부모요소의 컨텐츠너비가 800px로 줄어든다면, 한 줄에는 2개의 자녀요소가 배치되고(800 / 300 = 2.66), 각 자녀요소의 너비는 366.66px이 되며(300 + (200 / 3) = 366.66px), 3번째 자녀요소는 아랫줄로 내려간다.

주요 변수

  • --grid-min-item-size: 자녀요소의 최소너비.
  • --grid-space: 자녀요소들 간의 간격.
  • data-equal-row: 요소의 어트리뷰트로 추가하면 모든 행의 높이가 같아진다.

Switcher와의 차이점

Switcher

  • 모든 자녀요소를 가로로 배열하거나 아니면 모두 세로로 배열한다.
  • 가로배열과 세로배열 간의 전환을 부모요소의 컨텐츠너비에 기준한다.

Grid

  • 자녀요소를 x개씩 가로로 배열하고, 여남은 자녀요소들은 다음 줄에 배열한다.
  • 가로로 배열되는 자녀요소의 갯수를 자녀요소의 최소너비에 기준한다.
